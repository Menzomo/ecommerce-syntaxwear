# Syntaxwear - E-commerce de Sneakers e Tênis Online

Bem-vindo ao **Syntaxwear**, uma plataforma moderna de e-commerce focada na venda de sneakers e tênis de alta qualidade. O projeto foi desenvolvido com uma arquitetura limpa e modular, priorizando a experiência do usuário e a responsividade.

---

## 🚀 Sobre o Projeto

O Syntaxwear é um site de compras online projetado para amantes de calçados que buscam estilo, conforto e inovação. Com uma interface visualmente atraente ("Krypton One"), o site oferece navegação por categorias e uma vitrine de produtos em destaque.

## ✨ Funcionalidades

- **Design Responsivo:** Navegação adaptável para dispositivos móveis (menu hambúrguer) e desktops.
- **Navegação por Categorias:** Áreas dedicadas para calçados **Casuais, Esportivos, Modernos e Futuristas**.
- **Seção Hero de Impacto:** Chamada visual para coleções exclusivas.
- **Grid de Produtos:** Exibição organizada de produtos em destaque.
- **Newsletter:** Integração no rodapé para captura de e-mails.
- **Integração Social:** Links para Instagram, WhatsApp, TikTok e Facebook.
- **Rodapé Completo:** Links rápidos para atendimento, sobre a empresa e categorias.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas de desenvolvimento web front-end:

- **HTML5:** Estrutura semântica para melhor acessibilidade e SEO.
- **CSS3 (Modular):**
  - **Reset:** Para consistência entre navegadores.
  - **Variáveis:** Uso de `variables.css` para facilitar a manutenção de cores e fontes.
  - **Componentização:** Estilos divididos por componentes (`header.css`, `hero.css`, `footer.css`, etc.).
- **JavaScript:** Lógica client-side (preparado para expansão de funcionalidades).

## 📁 Estrutura de Pastas

```text
ecommerce-syntaxwear/
├── css/

├── images/
│   ├── banners/          # Imagens de fundo e hero
│   ├── icons/            # Ícones de interface (cart, user, menu)
│   ├── logo/             # Identidade visual da marca
│   └── products/         # Galeria de fotos dos produtos
├── js/
│   └── main.js           # Lógica principal do site
└── index.html            # Ponto de entrada do site
```

## 🚀 Como ter acesso ao Projeto

1.  **Clone o repositório:**
    ```bash

    ```
2.  **Navegue até a pasta:**
    ```bash
    cd ecommerce-syntaxwear
    ```
3.  **Abra o arquivo `index.html`:**
    Basta abrir o arquivo diretamente em seu navegador preferido ou utilizar a extensão **Live Server** no VS Code para uma melhor experiência de desenvolvimento.
